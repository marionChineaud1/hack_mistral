## Working product description

North is an AI investigative journalist for viral information. For the hackathon MVP, a user pastes an **X post URL** into Mistral Vibe. Specialised MCP tools collect evidence and context, analyse how the claim is framed, then produce a transparent investigation rather than a binary fact-check.

> AI should not tell people what to think. It should help them understand why something deserves their attention.
